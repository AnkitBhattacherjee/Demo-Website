# Demo Website — UI Project

## Overview

This repository contains the demo website built using **HTML** and **Tailwind CSS**.
It showcases responsive layout, modern UI components, and fashion-style design patterns.

## Tech Stack

* HTML5
* Tailwind CSS
* Remix Icons
* Responsive design

## How to Run

Simply open:

index.html

in any modern browser.

No server setup required.

## Development Setup (Optional)

If you want to rebuild Tailwind styles:

npm install
npx tailwindcss -i style.css -o output.css --watch

## Project Structure

* /assets → images & media
* index.html → main page
* output.css → compiled Tailwind styles

## Notes

This is a front-end demo project prepared for UI/UX presentation and component showcase.
